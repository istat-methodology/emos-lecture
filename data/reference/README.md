# Reference Data

This folder contains classification information used as the target knowledge base.

## `ateco22_disentangled.csv`

Short textual descriptors for ATECO 2022 classes. In the lab, these descriptors represent the possible codes that the automatic coding system can suggest.

## `ateco22_classification.csv`

Fuller ATECO metadata. It is useful for explanations and later extensions, but the first notebook mainly uses the shorter descriptor file.
