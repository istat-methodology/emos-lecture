# Sample Data

The teaching notebook uses `labelled_queries_sample.csv`, a small reproducible sample from `data/raw/labelled_queries.csv`.

The sample was created with these choices:

- keep multi-word queries
- remove duplicated query/code pairs
- sample 500 rows with random seed 42
- write true codes in dotted ATECO style, for example `73.11.0`

The raw data remains available, so students can later repeat the exercise with a larger sample.
